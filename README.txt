Приложение состоит из 3х страниц. Страница авторизации index.html, страница администратора adminList.html (с возможностью добавлять, изменять или удалять записи), страница пользователя userList.html (с возможностью осуществлять поиск по списку контактов).

Запуск:
1) Загрузить все файлы на сервер.
2) В файле script.js изменить путь к файлам adminList.html и userList.html в 27 и 31 строке в соответствии с настройками вашего сервера.
3) Запустить index.html
